CleanClip

Convert copied rich text into clean plain text and keep a short, private clipboard history that clears when the app quits.

Get Support Read Privacy Policy

CleanClip runs only from the macOS menu bar. It starts turned off, processes clipboard text on device, and does not require an account.

Plain Text

Remove unwanted fonts, colors, links, and formatting before pasting into notes, documents, editors, and issue trackers.

Memory Only

Recent clipboard history is kept in memory and clears when CleanClip quits. Clipboard history is not saved to disk.

Private by Default

No account, analytics, tracking, server upload, or third-party SDK is used by CleanClip.
